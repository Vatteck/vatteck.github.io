// CORE SYSTEMS

Four Sources, One Interface

📦

Multi-Source Switcher

Toggle between official Arch repositories, AUR, Flatpak, and AppImage — all from the same search bar. No context-switching between different tools.

🩺

System Health Cockpit

At-a-glance dashboard showing package health, available updates, orphaned packages, and disk usage. Catch issues before they break your system.

📝

PKGBUILD Viewer + Diffs

Inspect AUR PKGBUILDs before installing. View diffs between versions. No more blindly trusting `yay -S`.

🔍

Universal Preview

Hover any package to see description, version, size, dependencies, and install status — regardless of source. Consistent UX across all package types.
