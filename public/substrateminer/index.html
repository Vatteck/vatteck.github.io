HASHFACTORY

An idle-clicker tycoon game where you build and optimize a hardware-inspired mining operation. Formerly known as Substrate: Miner.

View on GitHub
// GAMEPLAY

Build Your Mining Empire

⛏️

Idle Progression

Your mining rigs keep working even when you're not tapping. Upgrade hardware components to boost hash rates and passive income.

⚡

Hardware-Inspired Systems

CPUs, GPUs, ASICs, cooling solutions, power supplies — each with meaningful trade-offs. Build a rig that reflects real hardware constraints.

🔄

Prestige System

Reset your operation to earn permanent multipliers. Each prestige run gets faster. Optimize the cycle to push deeper into the endgame.

🔢

Procedural Resource Generation

No two runs are identical. Resource outputs, hardware rolls, and market conditions shift — adapt your strategy accordingly.

// FEATURES

What's Inside

🖥️

Hardware-Inspired UI

Terminal aesthetic meets hardware dashboards. Monitor temps, clock speeds, and hash rates in real time.

📊

Backend Calculation Engine

Optimized math core runs the simulation. Resource generation, hardware degradation, and market prices all update deterministically.

🎯

Low-Latency Input

Responsive tapping and swiping. Every interaction feeds directly into the calculation pipeline with no perceptible lag.
