Systems

Deep Simulation, Not Scripted

🧬

SVG Avatar System

Modular character rendering: 7 eye styles, 18 hairstyles, 4 facial hair variants, aging wrinkles. Your character visibly ages across decades.

📜

Event-Sourced LifeLog

Every stat change, age-up, purchase, and decision is logged as an immutable event. Stats are projections computed from the trail — no fragile save states.

🎭

Yarn Narrative Engine

13 Yarn dialogue scripts power branching storylines. Context-aware narrative layers built from current character state, health, relationships, and recent events.

🎵

Dynamic Audio

5 background music tracks transition with game state. UI sounds for interactions, game-over sting. Powered by Audioplayers with background playback support.

✨

Animation-Rich UI

Rive vector animations, Flutter Animate transitions, shimmer loading states, confetti celebrations, animated digits and emoji. Every interaction has weight.

🔮

Supernatural & Political Overlays

Optional depth layers that layer onto any life: vampire covens with bloodline tracking, cult devotion mechanics, royal succession, political influence networks.

Built With

Architecture

🐦

Flutter + Dart

Cross-platform from one codebase. Android, iOS, Web, Windows, macOS, Linux.

🔄

Riverpod

Four provider patterns — singleton services, async data, real-time streams, parameterized family providers.

🗄️

Drift + SQLite

35 tables, schema v65. Idempotent migrations. Query-heavy — no wasteful writes.

🎬

Rive + Flutter Animate

Vector animation runtime paired with declarative animation library. No video assets.
